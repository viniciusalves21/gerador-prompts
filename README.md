# Gerador de Prompts - Lailla Agente

Este é um utilitário interno da **Lailla.io** para geração assistida de configurações e prompts para agentes de IA. A ferramenta permite personalizar a identidade, diretrizes de comunicação e objetivos dos agentes de forma modular e rápida.

## Tecnologias Utilizadas

- **Frontend:** HTML5, Tailwind CSS, JavaScript (Vanilla).
- **Ícones:** Lucide Icons.
- **Backend:** OpenAI API, Vercel Serverless Functions (Node.js).
